Сортировка: Список names сортируется по алфавиту.

Как работает программа:

Координаты ферзей задаются в двух списках: x и y.
Проверка пар: Программа проверяет каждую пару ферзей, чтобы определить, находятся ли они на одной горизонтали, вертикали или диагонали.
Условие атаки: Если хотя бы одна пара ферзей бьёт друг друга, переменная isAttacking становится true, и программа выводит YES. Если ни одна пара не бьёт друг друга, выводится NO.

Вы можете изменить значения в списках x и y, чтобы протестировать разные расстановки ферзей.
8
Объяснение:

Список паролей: В программе задан список паролей в хронологическом порядке.
Проверка на новую учётную запись: Для каждого пароля проверяется, не является ли он продолжением какого-либо из предыдущих паролей. Если не является, значит это новая учётная запись.
Подсчёт изменений: Для каждой учётной записи подсчитывается количество изменений пароля.
Максимальное количество изменений: Находится максимальное значение из всех подсчётов.
